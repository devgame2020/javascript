# 데이터 타입 - 문자열과 숫자
# 최신 ECMAScript 표준은 다음과 같은 7개의 자료형을  정의한다.

+ 기본 자료형 (Primitive) 인 여섯가지 데이터 타입
    + Boolean
    + Null
    + Undefined
    + Number
    + String  (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)
    + Symbol (ECMAScript 6 에 추가됨)
+ 별도로 Object 도 있음
